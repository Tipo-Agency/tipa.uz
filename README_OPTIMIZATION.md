# Автоматическая оптимизация изображений

## Что настроено

1. **Скрипт оптимизации** (`scripts/optimize-images.js`)
   - Автоматически оптимизирует все изображения в папке `public/`
   - Сжимает изображения до 85% качества (JPEG) или 90% (PNG)
   - Уменьшает размеры больших изображений (макс. 1920x1080)
   - Пропускает уже маленькие файлы (< 500KB)

2. **Автоматический запуск при билде**
   - Скрипт запускается автоматически перед `vite build`
   - Все изображения оптимизируются перед деплоем

3. **Компонент OptimizedImage**
   - Lazy loading
   - Blur placeholder
   - Responsive images

## Установка

```bash
npm install --save-dev sharp
```

## Использование

### Автоматически (при билде)
```bash
npm run build
```
Скрипт оптимизации запустится автоматически.

### Вручную
```bash
npm run optimize-images
```

## Что делает скрипт

- ✅ Находит все изображения в `public/`
- ✅ Проверяет размер каждого файла
- ✅ Оптимизирует файлы больше 500KB
- ✅ Сжимает до 85% качества (JPEG) или 90% (PNG)
- ✅ Уменьшает размеры больших изображений
- ✅ Показывает статистику оптимизации

## Рекомендации

1. Запускайте `npm run optimize-images` перед коммитом больших изображений
2. Используйте компонент `OptimizedImage` вместо обычных `<img>` тегов
3. Проверяйте размеры изображений перед загрузкой в репозиторий
